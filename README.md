# Blockchain-Based Job Recruitment System

A decentralized recruitment platform built on the Stacks blockchain using Clarity smart contracts. This system enables transparent, trustless hiring processes with verified resumes and conditional payment contracts.

## 🎯 Overview

The blockchain-based job recruitment system revolutionizes traditional hiring by leveraging blockchain technology to create a transparent, decentralized platform where job seekers can register verified work histories and employers can create trustless hiring contracts with conditional payments.

## ✨ Key Features

### Resume Registry
- **Decentralized Resume Storage**: Store and verify professional resumes and work history on the blockchain
- **Identity Verification**: Link resumes to verified blockchain addresses
- **Immutable Records**: Create tamper-proof work history records
- **Skills Validation**: Record and verify professional skills and certifications
- **Experience Tracking**: Maintain chronological work experience records

### Hiring Contract System
- **Smart Contract Jobs**: Create automated job offers with predefined terms
- **Conditional Payments**: Implement milestone-based payments and escrow functionality
- **Trustless Onboarding**: Automate the hiring process without intermediaries
- **Performance Metrics**: Track and verify job performance on-chain
- **Dispute Resolution**: Built-in mechanisms for handling contract disputes

## 🏗️ Architecture

The system consists of two main smart contracts:

1. **Resume Registry Contract** (`resume-registry.clar`)
   - Manages resume registration and verification
   - Handles work history records
   - Validates professional credentials

2. **Hiring Contract** (`hiring-contract.clar`)
   - Creates and manages job offers
   - Handles payment escrow and releases
   - Manages onboarding processes
